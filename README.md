# 🍓 Todo List

A cozy notebook-style To-Do List with a built-in Pomodoro Timer — all in a single HTML file, no installation needed.

[Try Here](https://baby-lm.github.io/my-todo-list/my-todo-list.html)

---

## ✨ Features

- 📝 Add, check, and delete tasks
- 🍅 Built-in Pomodoro Timer (focus / break sessions)
- 🎨 Warm cream & soft purple notebook aesthetic
- 💾 Saves your tasks automatically (localStorage)
- 📱 Works on mobile and desktop
- ⚡ No internet required after first load

---

## 🚀 How to Use

1. Open the app via the link above
2. Type a task and press **Enter** or click **+**
3. Click a task to mark it done ✓
4. Use the **Pomodoro Timer** on the right to stay focused
